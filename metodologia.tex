\subsection{Cenário}\label{sec:cenario}

O cenário detalhado adotado neste trabalho é baseado em \pool;
de consulta por classe monorrótulo; com amostragem um a um de uma distribuição estacionária com
custos uniformes; com oráculo único e consistente,
sendo um rótulo inicial por classe simuladamente obtido via
\engl{aprendizado guiado}{guided learning} \citep{conf/kdd/AttenbergP10};
com atributos sem valores faltantes;
e, sem redução do tamanho das bases de dados originais\footnote{Exceto para a estratégia de \eer,
para ser computacionalmente tratável.}.
